이제 내 서버를 터미널이 아닌 GUI를 통해 접속해봅시다.

서버를 열어 다음 명령어들을 순차적으로 입력해줍니다.

```
sudo apt-get update
sudo apt-get install ubuntu-desktop
```

``ubuntu-desktop``의 용량이 **2.1GB**이기 때문에 서버 용량을 잘 확인하신 후 설치하시기 바랍니다.
