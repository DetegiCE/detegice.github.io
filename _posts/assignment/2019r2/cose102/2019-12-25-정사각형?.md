# DESCRIPTION
주어진 4개의 점이 정사각형인지 체크하는 프로그램을 작성하세요.

# INPUT
* Line 1 : 테스트케이스 T (1~1,000)

* Line 2 ~ T+1 : 공백으로 구분된 4쌍의 좌표 값 (각 좌표값은 -100~100 범위의 정수) 

# OUTPUT
* Line 1 ~ T : 정사각형일 경우 square를 출력, 아닐 경우 not square를 출력

# SAMPLE INPUT
```
3
0 0 0 2 2 0 2 2
0 0 4 0 2 2 2 -2
0 0 0 2 2 2 2 4
```

# SAMPLE OUTPUT
```
square
square
not square
```
