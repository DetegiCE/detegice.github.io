# DESCRIPTION
주어진 숫자들을 조합하여 가장 크게 만들어 낼 수 있는 3의 배수를 출력하시오. (모든 숫자를 사용하지 않아도 됨)

 

# INPUT
* Line 1 : 테스트케이스 T (1~1,000)

* Line 2 ~ T+1 : 문자열 (공백으로 구분된 숫자; 숫자의 개수는 100개를 넘지 않음)

# OUTPUT
* Line 1 ~ T : 3의 배수

 - 3의 배수가 없을 경우 0을 출력

# SAMPLE INPUT
```
1
0 1 6 7 8
```

# SAMPLE OUTPUT
```
8760
```
