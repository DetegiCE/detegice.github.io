# DESCRIPTION
0부터 100사이의 정수를 입력받아 16진수로 출력하는 프로그램을 작성하세요.

Write a program that prompts the user to enter an integer between 0 and 100 and displays its corresponding hex number.

 

# INPUT
* Line 1 : 정수 (0~100)

 

# OUTPUT
* Line 1 : HEX값 (0~64범위의 hex값; a-f는 소문자로 표시)

 

# SAMPLE INPUT
```
100
```

# SAMPLE OUTPUT
```
64
```
