# DESCRIPTION
가로 n칸짜리 원고지에 1부터 m까지 순서대로 써보자.

이 때, 원고지에는 한 칸에 숫자 하나만 쓸 수 있다.

# INPUT
* Line 1 : 원고지의 가로 칸 수 n이 입력된다. (1≤n≤100)

* Line 2 : 써내려갈 숫자의 최댓값 m이 입력된다. (1≤m≤1000)

# OUTPUT
* Line 1~ : 가로 n칸짜리 원고지에 1부터 m까지 적은 결과를 출력한다.

# SAMPLE INPUT
```
5
15
```

# SAMPLE OUTPUT
```
12345
67891
01112
13141
5
```
