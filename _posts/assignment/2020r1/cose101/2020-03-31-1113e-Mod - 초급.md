**고급을 먼저 풀고 초급을 나중에 풀었기 때문에 코드가 동일하다**

# DESCRIPTION
A를 n번 곱한 값을 B로 나눈 나머지를 구해보자.

# INPUT
Line 1: 입력 갯수 N (1≤N≤10000)

Line 2~N+1: A, n, B가 공백으로 구분되어 입력된다. (1 ≤ A,n,B ≤ 1000)

# OUTPUT
* Line 1~N : A를 n번 곱한 값을 B로 나눈 나머지를 출력한다.

# SAMPLE INPUT
```
1
2 3 5
```

# SAMPLE OUTPUT
```
3
```
