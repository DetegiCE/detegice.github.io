# DESCRIPTION
자연수 n의 약수의 합을 출력해보자.

# INPUT
Line 1: 입력의 갯수 N (1≤N≤1000)

Line 2 ~ N+1: 자연수 n (1≤n≤5000)

# OUTPUT
Line 1~N: n의 약수의 갯수

# SAMPLE INPUT
```
3
24
36
47
```

# SAMPLE OUTPUT
```
60
91
48
```
