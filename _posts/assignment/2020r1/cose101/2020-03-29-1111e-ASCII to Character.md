# DESCRIPTION
정수를 입력받아서 아스키코드 문자로 출력해보자.

참고로 아스키코드표는 다음과 같다.

[아스키코드표](https://detegice.github.io/ascii-code/)

# INPUT
* Line 1 : 정수 d가 주어진다.

# OUTPUT
* Line 1 : d에 해당하는 아스키코드 문자를 출력한다.

# SAMPLE INPUT
~~~
97
~~~

# SAMPLE OUTPUT
~~~
a
~~~
