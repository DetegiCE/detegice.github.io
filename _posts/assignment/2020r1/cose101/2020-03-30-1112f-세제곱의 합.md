# DESCRIPTION
$1^3+2^3+...n^3$을 구해보자.

# INPUT
* Line 1: 입력의 갯수

* Line 2~N+1 : n이 주어진다.

# OUTPUT
* Line 1~N : $1^3+2^3+...n^3$의 값을 구한다.

# SAMPLE INPUT
```
2
3
5
```

# SAMPLE OUTPUT
```
36
225
```

# HINT
1+8+27=36
