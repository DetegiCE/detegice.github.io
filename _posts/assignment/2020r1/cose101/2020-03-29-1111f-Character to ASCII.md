# DESCRIPTION
문자를 입력받아서 아스키코드 값으로 출력해보자.

참고로 아스키코드표는 다음과 같다.

[아스키코드 표](http://detegice.github.io/ascii-code)

# INPUT
* Line 1 : 문자 c가 주어진다.

# OUTPUT
* Line 1 : c의 아스키코드 값을 출력한다.

# SAMPLE INPUT
```
a
```

# SAMPLE OUTPUT
```
97
```
