# ASCIIコードとは？

ASCIIは、American Standard Code for Information Interchange(アメリカ情報交換用標準コード)を表します。
コンピュータは数字しか理解できないため、ASCIIコードは文字の数値表現です。

Ctrl + Fで文字を検索することができます。
