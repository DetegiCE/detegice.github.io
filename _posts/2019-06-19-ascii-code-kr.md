# 아스키 코드란?

ASCII(아스키)는 American Standard Code for Information Interchange(미국정보교환표준부호)의 줄임말 입니다.
영문 알파벳을 사용하는 대표적인 문자 인코딩으로, 컴퓨터와 통신 장비를 비롯한 문자를 사용하는 많은 장치에서 사용됩니다.

Ctrl+F로 문자를 검색하실 수 있습니다.
